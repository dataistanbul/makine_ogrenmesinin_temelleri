İçerik Üzerinden Puanlama¶

Problemin Formulleştirilmesi¶

Her bir kullanc iin katsays ($\theta$) renilir. Ardndan bu katsaylar ile niteliklerin (x) deerleri arplarak tahmini puanlar hesaplanr.

Forml : $ (\theta^j)^T *x_i $

$r^{(i,j)}$ : Eer kullanc filmi puanlad ise 1

$y^{(i,j)}$ : indis (i,j)=1 iken verilen puann deeri

$\theta^{(j)}$ : j kullanc iin katsay vektr

$x^{(i)}$ : i filmi iin nitelik vektr

Her bir kullanc iin tahmini puan $ (\theta^{(j)})^T *x_i$

$ m^{(j)} $: $j$ kullancs tarafndan puanlanan toplam film adedi

j kullancs iin katsay($\theta_j$) hesaplanmas :

$min_{\theta^{(j)}} 1/2 \begin{equation}\sum_{i:r(i,j)=1} ((\theta^j)^T *x_i - y^{(i,j)})^2 \end{equation} $

$min_{\theta^j} 1/2 \begin{equation}\sum_{i:r(i,j)=1} ((\theta^j)^T *x_i - y^{(i,j)})^2 \ + \lambda/2 \sum_{k=1}^n (\theta_k^j)^2 \end{equation} $

Optimizasyon Hedefi¶

* $ \theta^{(j)} $ katsaysn j kullancs iin hesaplamak iin kullanlacak forml :

$min_{\theta^j} 1/2 \begin{equation}\sum_{i:r(i,j)=1} ((\theta^j)^T *x_i - y^{(i,j)})^2 \ + \lambda/2 \sum_{k=1}^n (\theta_k^j)^2 \end{equation} $

* $ \theta^{(1)} ,\theta^{(2)} ,\theta^{(3)},...,\theta^{(n_u)} $ katsaylarn iin hesaplamak iin kullanlacak forml :

$min_{\theta^{(1)} ,..,\theta^{(n_u)}} 1/2 \begin{equation}\sum_{i:r(i,j)=1} ((\theta^j)^T *x_i - y^{(i,j)})^2 \ + \lambda/2 \sum_{j=1}^{n_u} \sum_{k=1}^n (\theta_k^j)^2 \end{equation} $

Optimizasyon Algoritması¶

\begin{equation} J_{(\theta^{(1)} ,..,\theta^{(n_u)})}=1/2*\sum_{i:r(i,j)=1} ((\theta^j)^T *x_i - y^{(i,j)})^2 \ + \lambda/2 \sum_{j=1}^{n_u} \sum_{k=1}^n (\theta_k^j)^2 \end{equation}

Dereceli Azaltma:

\begin{equation}\ \theta_k^{(j)} := \theta_k^{(j)}-a*\sum_{i:r(i,j)=1} ((\theta^{(j)})^T *x^{(i)} - y^{(i,j)})*x^{(i)} \ \ \ \ \ (k=0 \ \ iin) \end{equation}

\begin{equation}\ \theta_k^{(j)} := \theta_k^{(j)}-a*(\sum_{i:r(i,j)=1} ((\theta^{(j)})^T *x^{(i)} - y^{(i,j)})*x^{(i)} + \lambda*\theta_k^{(j)} )\ \ \ \ \ (k\neq0 \ \ iin) \end{equation}

\begin{equation}\ \frac{\partial J(\theta_0,\theta_1,...\theta_n)}{\partial \theta_k^{(j)}}=(\sum_{i:r(i,j)=1} ((\theta^{(j)})^T *x^{(i)} - y^{(i,j)})*x^{(i)} + \lambda*\theta_k^{(j)} )\ \end{equation}

Nitelik için Optimizasyon Algoritması¶

$x^{(i)}$ niteliklerini $\theta_0,\theta_1,...\theta_n$ bilinirken iin hesaplamak iin kullanlacak forml :

\begin{equation} min_{(x^{(i)})} \ \ 1/2*\sum_{i:r(i,j)=1} ((\theta^j)^T *x_i - y^{(i,j)})^2 \ + \lambda/2 \sum_{k=1}^n (x_k^{(i)})^2 \end{equation}

$x^{(1)},x^{(2)},x^{(3)},..,x^{(m)}$ niteliklerini $\theta_0,\theta_1,...\theta_n$ bilinirken iin hesaplamak iin kullanlacak forml :

\begin{equation} min_{ x^{(1)},x^{(2)},x^{(3)},..,x^{(m)} } \ \ 1/2*\sum_{i:r(i,j)=1} ((\theta^j)^T *x_i - y^{(i,j)})^2 \ + \lambda/2 \sum_{i=1}^{n_m} \sum_{k=1}^n (x_k^{(i)})^2 \end{equation}

Sonuç olarak : İşbirliğine Dayalı Filtreleme (collaborative filtering)¶

$x^{(1)},x^{(2)},x^{(3)},..,x^{(m)}$ niteliklerini biliyorsak , $\theta_0,\theta_1,...\theta_n$ katsaylarn hesaplayabiliriz.

$\theta_0,\theta_1,...\theta_n$ katsaylarn biliyorsak , $x^{(1)},x^{(2)},x^{(3)},..,x^{(m)}$ niteliklerini hesaplayabiliriz

Tahminimiz : $\theta -> x->\theta -> x->\theta -> x->\theta -> ... $ eklinde olacaktr.

İşbirliğine Dayalı Filtreleme (collaborative filtering) Optimizasyon Hedefi¶

$x^{(1)},x^{(2)},x^{(3)},..,x^{(m)}$ niteliklerini bilirken $ \theta^{(1)} ,\theta^{(2)} ,\theta^{(3)},...,\theta^{(n_u)} $ katsaylarn iin hesaplamak iin kullanlacak forml :

$min_{\theta^{(1)} ,..,\theta^{(n_u)}} 1/2 \begin{equation}\sum_{i:r(i,j)=1} ((\theta^j)^T *x_i - y^{(i,j)})^2 \ + \lambda/2 \sum_{j=1}^{n_u} \sum_{k=1}^n (\theta_k^j)^2 \end{equation} $

$ \theta^{(1)} ,\theta^{(2)} ,\theta^{(3)},...,\theta^{(n_u)} $ katsaylarn bilirken $x^{(1)},x^{(2)},x^{(3)},..,x^{(m)}$ niteliklerini iin hesaplamak iin kullanlacak forml :

$ min_{ x^{(1)},x^{(2)},x^{(3)},..,x^{(m)} } 1/2* \begin{equation} \sum_{i:r(i,j)=1} ((\theta^j)^T *x_i - y^{(i,j)})^2 \ + \lambda/2 \sum_{i=1}^{n_m} \sum_{k=1}^n (x_k^{(i)})^2 \end{equation} $

Ayn anda $ \theta^{(1)} ,\theta^{(2)} ,\theta^{(3)},...,\theta^{(n_u)} $ ve $x^{(1)},x^{(2)},x^{(3)},..,x^{(m)}$ minimize etmek iin :

$ min_{( x^{(1)},..,x^{(m)}) ,(\theta^{(1)} ,..,\theta^{(n_u)}) } 1/2* \begin{equation} \sum_{i:r(i,j)=1} ((\theta^j)^T *x_i - y^{(i,j)})^2 \ + \lambda/2 \sum_{i=1}^{n_m} \sum_{k=1}^n (x_k^{(i)})^2+ \lambda/2 \sum_{j=1}^{n_u} \sum_{k=1}^n (\theta_k^j)^2 \end{equation} $

İşbirliğine Dayalı Filtreleme (collaborative filtering) Algoritması¶

1. $x^{(1)},x^{(2)},x^{(3)},..,x^{(m)}$ , $ \theta^{(1)} ,\theta^{(2)} ,\theta^{(3)},...,\theta^{(n_u)} $ parametrelerine ufak rastgele saylar atanr .

2. $ J( x^{(1)},..,x^{(m)}) ,(\theta^{(1)} ,..,\theta^{(n_u)}) $ deerini minimize etmek iin "dereceli azaltma" veya "ileri optimizasyon" kullanlr. Bu parametrelere balangta ufak rastgele saylar atanr .

\begin{equation}\ x_k^{(i)} := x_k^{(i)}-a*(\sum_{i:r(i,j)=1} ((\theta^{(j)})^T *x^{(i)} - y^{(i,j)})*\theta_k^{(j)} + \lambda*x_k^{(i)})\ \end{equation}

\begin{equation}\ \theta_k^{(j)} := \theta_k^{(j)}-a*(\sum_{i:r(i,j)=1} ((\theta^{(j)})^T *x^{(i)} - y^{(i,j)})*x^{(i)} + \lambda*\theta_k^{(j)} )\ \end{equation}

3. $\theta$ katsaysna sahip kullanc ve $x$ nitelikleri hesaplanm bir film iin tahmini fill puan $(\theta^T *x)$'e eittir.

def bedel(param,y_norm,n,lambda_):
    nm=y.shape[0]
    nu=y.shape[1]
    r=-np.isnan(y)*1
    X=param[:nm*n].reshape(nm,n) 
    theta_=param[nm*n:].reshape(n,nu)
    J= (1/2)*np.sum(np.sum(np.multiply(r,((np.dot(X,theta_)-y))**2)))+(lambda_/2)*np.sum(np.sum(np.multiply(X,X)))+(lambda_/2)*np.sum(np.sum(np.multiply(theta_,theta_)))
    return J

def derece(param,y_norm,n,lambda_):
    
    nm=y.shape[0]
    
    nu=y.shape[1]
    
    r=-np.isnan(y_norm)*1
    
    y_norm=y_norm.fillna(0)
    
    X=param[:nm*n].reshape(nm,n) 

    theta_=param[nm*n:].reshape(n,nu)
    
    X_derece =  np.dot(np.multiply(r,((np.dot(X,theta_)-y_norm))) ,theta_.T)+  lambda_*X
    
    theta_derece =  np.dot(X.T, np.multiply(r,((np.dot(X,theta_)-y_norm))))+  lambda_*theta_
    
    derece_=np.array(np.concatenate(X_derece).tolist()+np.concatenate(theta_derece).tolist())
    
    return derece_
